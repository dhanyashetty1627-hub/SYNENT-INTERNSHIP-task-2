# Iris Data Visualization

## Problem Statement
The objective of this project is to perform Exploratory Data Analysis (EDA) and visualize the Iris dataset to understand patterns, distributions, and relationships among different flower species.

## Dataset Details
- Dataset: Iris Dataset
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
  - Species

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations Performed
1. Bar Chart
   - Displayed the count of each Iris species.

2. Histogram
   - Showed the distribution of sepal length values.

3. Scatter Plot
   - Visualized the relationship between petal length and petal width across species.

## Observations
- All species are equally represented in the dataset.
- Most flowers have sepal lengths between 5 cm and 7 cm.
- Petal length and petal width show a positive relationship.
- Setosa species is clearly separated from the other two species.

## Conclusion
Data visualization helps uncover hidden patterns and relationships within datasets. The Iris dataset demonstrates how graphical analysis can effectively distinguish species and reveal feature correlations.
